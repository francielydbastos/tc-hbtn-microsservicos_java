O Spring Boot é uma framework que facilita o desenvolvimento de aplicações web e microsserviços através de diversos módulos para gerenciamento de dados, estratégias para lidar com falhas, segurança e, principalmente, sua autoconfiguração. Outra grande vantagem do Spring Boot é já ter um servidor Tomcat, Jetty ou Undertow embutido, o que facilita na hora de testar e fazer deploy.
Além disso, o Spring Initializer permite simplificar a criação inicial da aplicação ao prover um "esqueleto" completo com várias dependências já selecionadas, o que facilita o desenvolvimento posterior. Caso seja necessário utilizar dependências adicionais, basta adicionar suas declarações no arquivo POM.xml, assim como é feito nos projetos Maven.
Na minha opinião, as melhores IDEs para se desenvolver projetos em Spring Boot é o STS - Spring Tool Suite. Como o próprio nome já diz, o STS já é uma IDE dedicada a projetos do framework Spring e conta com várias facilidades na hora de desenvolver projetos em Spring Boot.
